Chapter 2 Sets

Sets are a collection of distinct elements, below is a reference for set notation

\(\cap\) - Union

\(\in\) - belongs to

\(\varnothing\) - null set

\(\Omega\) - universal set

\(\omega\) - atom, singleton

\(\subset\) - proper subset, \(S\) is contained within \(A\), sometimes denoted even when S = A

\(\subseteq\) - subset, if \(S\) is contained within \(A\).
